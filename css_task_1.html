Style this site!

I want you to style this site. To do so you will go through a series of challenges.
All challenges are achievable only by editing the style tag in the head and nothing else.

To achieve this I want you to not touch the html and only edit the CSS inside the style tag. There might be styles that are in the html elements but you must not edit them.

...Ooookay so are you ready?

  • First: I want you to style all special text (all elements that have the class "special-text") with purple color.
  • Second: I want you to style all important text (all elements that have the class "important-text") with red color and make it bold.
  • Third: I want you to style all unordered list items (ul li) to have a list style of square instead of the default circle.
  • Fourth: I want you to style the body to have:
    1. Height of 100 percent of the view height
    2. Width of 100 percent of the view width
    3. Have a background color of light gray.
    4. Have a margin of 0 on all sides.
  • Fifth: I want you to style the element with id main-content to:
    1. Have width of 50 percent.
    2. Have a height of 100%
    3. Have a white background color
    4. Have a padding of 25px
    5. Margin on all sides to be automatically calculated. Because the width is 50 percent of the available space the margin will be automatically sized both on the left and right. This will have the effect of centering the content of a block element.
  • Sixth: I want you to style all headings to:
    1. Have their text centered
    2. Have their font family be Helvetica
    3. Have their color be darkslategray
    4. Only h1 should also be with a font size of 32pt
  • Seventh: The last paragraph should be styled with centered text content.
  • Eight: The image should have a 2px solid light gray border.
